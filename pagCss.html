CSS

¿QUÉ ES CSS?


Definición



CSS (siglas en inglés de Cascading Style Sheets), en español "Hojas de estilo en cascada", un lenguaje de marcas enfocado a definir, crear y mejorar la presentación de un documento basado en HTML. Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.



Se les denomina hojas de estilo «en cascada» porque puedes tener varias hojas y una de ellas con las propiedades heredadas (o «en cascada») de otras.

Historia - ¿Por qué surgió?



Durante su evolución el lenguaje de marcas HTML ha ido permitiendo cierto grado de personalización. Desde poner en negrita un texto, cambiar su color hasta alinear una fotografía, ésto se realizaba casi de manera artesanal, añadiendo línea a línea etiquetas que ayudaban a su diseño.



Sin embargo, con el paso de los años, dió como consecuencia que cualquier proyecto web se viera idéntico y ambiguo. Por ello, se creó CSS, una solución para aquellos que querían realizar cambios en la apariencia de una web sin afectar de manera dramática su contenido.

¿PARA QUÉ SIRVE?


Funcion



Con CSS puedes crear reglas para decirle a tu sitio web cómo quieres mostrar la información y guardar los comandos para elementos de estilo (como fuentes, colores, tamaños, etc.) separados de los que configuran el contenido.


En otras palabras, sirve para darle formato y diseño a tu página WEB.

ETIQUETA DE ESTILO


¿Qué es una etiqueta de estilo?



La etiqueta <style> dentro del código HTML permite añadir estilos al contenido del documento HTML. Un ejemplo de estilo es el color de un texto, su tamaño, la separación entre líneas, etc.


¿Cómo llamar una etiqueta



Para llamar a un elemento con una clase CSS específica, escribiríamos el nombre de la clase precedida por un punto (.).
